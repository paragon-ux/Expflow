# reproduction

**Gate C implementation.** This directory owns regeneration attempts, equivalence evaluations, and structural reuse results.

Unknown regeneration outcomes remain durable records. Equivalence is an attributed evaluation, not an inferred fact. Structural reuse is gated by explicit license and authority-policy checks and does not transfer completion, verification, or authority from the source workflow.

This directory does not run external tools, call models, inspect adapters, reconcile lost responses, or execute generated code.
